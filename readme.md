## Projet Machine Learning II

L’objectif de ce projet est de se familiariser avec l’utilisation de réseaux de neurones convolutionnels pré-entraînés. Ici, on utilisera le réseau VGG16 pour effectuer de la classification d’images. Nous étudions ici différentes approches basées sur le transfert learning et l’extraction de features.

## Structure du Projet

Le dépôt est organisé de la manière suivante :
Project.ipynb Cœur du projet. C'est le notebook principal qui génère les résultats et les visualisations.
data/ Contient tous les jeux de données nécessaires à l'exécution.
src/ Regroupe le code source, les fonctions utilitaires et les modules Python.

## Installation & Utilisation

1. Prérequis

Assurez-vous d'avoir Python installé (version 3.10 recommandée) ainsi que Jupyter Notebook ou JupyterLab. 2.
Les dépendances ne sont pas indiqué dans un requirement mais sont les dépendances classique d'un projet IA
Pour voir les résultats et utiliser le projet, il n'y a qu'une seule étape :

    Ouvrez le fichier Project.ipynb à la racine du projet.
    Exécutez toutes les cellules (Menu Cell > Run All).
    Note : Le notebook importe automatiquement les modules situés dans le dossier src et récupère les données dans data.
